SOAPBOX:
まだプレゼンに PowerPoint・Keynote・PDF 使ってるの?

権藤研 Lightning Talk 2021/04/20
新山

1. なぜ PowerPoint・Keynote・PDF は悪か

  • 作成するのに専用アプリが必要。
  • 仕様がオープンでない。
  • サイズを調節できない。
  • 動作が遅い。
  • マウスを使わないと作業できない。
  • 今どきページ単位で表示?

2. じゃあどうすれば?

HTMLを使え。
  • 誰でもブラウザさえあれば見れる。共有も簡単。
  • テキストエディタさえあれば作れる。
  • 内容が連続して見れるし、現在位置もわかりやすい。
  • 拡大・縮小で自動リフロー。
  • 音声読み上げ可。
  • 画像でも表でもJSでもYouTubeでも埋め込み可。
  • Alternative: Markdown.

3. でも…

  • 「図が入れられない」 → SVGって知ってる? 手書き、 SVG-edit、 diagrams。
  • 「数式が入れられない」 → MathJax、 KaTeX。
  • 「プレゼンテーション用のJSフレームワークが…」 → 必要なし。 このファイル自身をパクれ。
  • 「アニメーションが使えない」 → カッコつけるのは100年早い。
  • 「HTMLでプレゼンやってる奴なんている?」 → おります。

4. それでも人が PowerPoint・Keynote を使う理由

  • プレゼンがTED並にスゴい。
  • 金と時間が余っている。
  • 記憶力が低下しててHTMLの20個のタグが覚えられない。
    (プログラムはScratchで作ってるんでちゅか?)

5. 結論

あなたのプレゼンはダサいし、金も時間もない。 しかし、あなたにはまともな技術を覚えて使いこなす能力がある。 アホなことで時間を無駄にするのはおやめなさい。
Yusuke Shinyama
