3. FGyama の構造

  • 制御フローおよびデータフローを同時に表現。
  • 各変数を個別に扱う。
  • データの依存関係を重視する。

例

int max(int[] a) {
    int y = -1;
    for (int x : a) {
        if (y < x) {
            y = x;
        }
    }
    return y;
}
