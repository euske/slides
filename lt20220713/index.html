SOAPBOX:
Are you STILL using PowerPoint/Keynote/PDF for presentation??!1

Shinoda Lab Lightning Talk 2022/07/13
Yusuke Shinyama

1. Why PowerPoint/Keynote/PDF is Evil

  • Non-open format.
  • Needs a proprietary app to create.
  • Slow.
  • Requires a mouse.
  • No auto-adjust for screen size.
  • Are we still doing page-based presentation in 2022?

2. What to Do, Then?

Use HTML.
(or Markdown)
  • Anyone can see it with a browser.
  • Even blind people can see it with a screen reader!
  • Only requires a text editor to create.
  • Allows continuous scroll. Easy to see where we are at →
  • Responsive (auto-adjust for screen size).
  • Easy to share: euske.github.io
  • You can embed whatever you want.
      Answer:  
    A little MNIST demo.

3. But...

  • "How to put a figure?"
    Ever heard of SVG?
    It can be hand-written, or made by apps like SVG-edit or diagrams.
  • "How to put a formula?"
    Use MathJax or KaTeX.
  • "How to choose the best HTML template?"
    No need. Just copy this file.
  • "How to do animation?"
    Animated slides are bad idea anyway.

4. You still want to use PowerPoint/Keynote because...

  • You are going to speak at TED.
  • You have too much time and money.
  • You are too old to remember 20+ HTML tags.
    (I wonder how you've ever learned a programming language.)

5. Conclusion

Don't be lazy. You have an ability to learn and use good stuff.

And that's thoughtfulness.
