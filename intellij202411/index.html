2. コードの編集

2.1. Mac基本

  • 行頭・行末:
    • Command ⌘ + ←
    • Command ⌘ + →
  • １単語前・１単語後
    • Option ⌥ + ←
    • Option ⌥ + →
  • 文字選択
    • Shift + ←
    • Shift + →
  • 文字選択 + 行頭・行末
    • Shift + Command ⌘ + ←
    • Shift + Command ⌘ + →
  • 文字選択 + １単語前・１単語後
    • Shift + Option ⌥ + ←
    • Shift + Option ⌥ + →
  • 単語選択
    • ダブルクリック
  • 単語範囲選択
    • ダブルクリック + ドラッグ
  • 行選択
    • トリプルクリック
  • 行範囲選択
    • トリプルクリック + ドラッグ

2.2. IntelliJの便利機能

  • 補完
    • Tab
  • 1行コピー
    • (選択せずに) Command ⌘ + C
  • 1行切り取り
    • (選択せずに) Command ⌘ + X
  • 検索
    • Command ⌘ + F
  • ファイル横断検索
    • Shift + Command ⌘ + F
  • なんでも検索
    • Shift Shift
  • 折りたたみ・展開
    • Command ⌘ + -
    • Command ⌘ + +
  • 一斉折りたたみ
    • Shift + Command ⌘ + -
  • マルチカーソル
    • Option ⌥ Option ⌥ + ↓
  • 検索 + マルチカーソル
    • Control ⌃ + G
  • 定義にジャンプ
    • Command ⌘ + クリック
  • 前の場所に戻る
    • Option ⌥ + Command ⌘ + ←
  • 一括整形
    • Option ⌥ + Command ⌘ + L
  • なんでも問題解決
    • Option ⌥ + Return
